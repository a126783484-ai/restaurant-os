# Mobile UX

MVP standard: usable, not premium redesign.

- Login/register: no horizontal overflow, readable errors, 44px+ controls.
- Orders: create flow usable on phone; submit disabled while pending.
- Order detail: status, payment, quantity edits are touch-friendly.
- KDS: tablet/phone columns stack cleanly.
- Dashboard: never blank; show loading, empty, or readable error states.
